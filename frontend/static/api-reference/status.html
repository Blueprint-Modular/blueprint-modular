Status

Indicateurs de chargement et retours : bpm.spinner, bpm.progress, bpm.toast, bpm.status, bpm.exception.

Référence API

FonctionBPM
spinner()bpm.spinner()
progress()bpm.progress()
status()bpm.status()
toast()bpm.toast()
balloons()bpm.balloons()
snow()bpm.snow()
exception()bpm.exception()

Exemples

import bpm
import time

with bpm.spinner("Chargement..."):
    time.sleep(2)

bpm.progress(0.75, text="75 %")
bpm.toast("Terminé !")
bpm.balloons()
bpm.snow()

with bpm.status("Traitement en cours"):
    bpm.write("Étape 1...")
    bpm.write("Étape 2...")

try:
    raise ValueError("Erreur exemple")
except Exception as e:
    bpm.exception(e)
