First app

Tutoriel : du vide à une app fonctionnelle en quelques minutes.

1. Créer un fichier app.py

import bpm

bpm.title("Ma première app BPM")

2. Ajouter une métrique et un tableau

import bpm
import pandas as pd

bpm.title("Ma première app BPM")
df = pd.DataFrame({"A": [1, 2, 3], "B": [4, 5, 6]})
bpm.metric("Lignes", len(df))
bpm.table(df)

3. Ajouter un toggle qui conditionne l'affichage

if bpm.toggle("Afficher le tableau"):
    bpm.table(df)

4. Lancer en local

bpm run app.py

Ouvrez l'URL affichée (ex. http://localhost:8501).

5. Accessible en réseau

Par défaut, le serveur écoute sur toutes les interfaces. Sur le même réseau, d'autres machines peuvent accéder à l'app via l'IP de votre machine.
