markdown

Affiche du contenu rendu en Markdown (gras, italique, listes, liens, etc.).

Référence API

FonctionBPM
markdown()bpm.markdown()

Exemple

import bpm

bpm.markdown("Du **gras** et de l'*italique*.")
bpm.markdown("# Titre\\n- liste\\n- à puces")
