Status

Indicateurs de chargement et retours : bpm.spinner, bpm.progress, bpm.toast, bpm.status, bpm.exception.

Mapping Streamlit → BPM

StreamlitBPM
st.spinner()bpm.spinner()
st.progress()bpm.progress()
st.status()bpm.status()
st.toast()bpm.toast()
st.balloons()bpm.balloons()
st.snow()bpm.snow()
st.exception()bpm.exception()

Exemples

import bpm
import time

with bpm.spinner("Chargement..."):
    time.sleep(2)

bpm.progress(0.75, text="75 %")
bpm.toast("Terminé !")
bpm.balloons()
bpm.snow()

with bpm.status("Traitement en cours"):
    bpm.write("Étape 1...")
    bpm.write("Étape 2...")

try:
    raise ValueError("Erreur exemple")
except Exception as e:
    bpm.exception(e)
