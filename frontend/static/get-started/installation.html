Installation

⚠️
Important — Blueprint Modular n'est pas encore publié sur PyPI. Les commandes ci-dessous décrivent l'objectif final. Pour tester dès maintenant, voir « Installation depuis le repo ».

Prérequis

  • Python 3.9+
  • pip à jour : pip install --upgrade pip
  • Optionnel : environnement virtuel recommandé (python -m venv .venv)

Installation (objectif — pas encore disponible)

pip install blueprint-modular

Installation depuis le repo GitHub (disponible maintenant)

git clone https://github.com/remigit55/blueprint-modular
cd blueprint-modular
pip install -e .

Sur Linux/macOS, en cas d'erreur « externally-managed-environment », créez un venv : python3 -m venv .venv, puis source .venv/bin/activate et pip install -e .

Lancer une app

bpm run app.py

L'app est servie sur http://localhost:8501 (port et host configurables avec --port et --host).

Scaffolder une app vide

bpm init

Génère app.py, requirements.txt et app.config.js (structure minimale pour démarrer).

Vérifier l'installation

bpm --version
