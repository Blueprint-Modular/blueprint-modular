Layout

Organisation de l'interface : bpm.sidebar, bpm.columns, bpm.tabs, bpm.expander, bpm.form.

Référence API

FonctionBPM
sidebarbpm.sidebar
columns()bpm.columns()
tabs()bpm.tabs()
expander()bpm.expander()
popover()bpm.popover()
container()bpm.container()
form()bpm.form()
space()bpm.space()

Exemples

import bpm

with bpm.sidebar:
    bpm.select("Page", ["Accueil", "Données"])

col1, col2, col3 = bpm.columns(3)
with col1: bpm.metric("A", 1)
with col2: bpm.metric("B", 2)
with col3: bpm.metric("C", 3)

tab1, tab2 = bpm.tabs(["Vue 1", "Vue 2"])
with tab1: bpm.write("Contenu 1")
with tab2: bpm.write("Contenu 2")

with bpm.expander("Voir plus"):
    bpm.write("Contenu repliable.")

with bpm.form("mon_form"):
    bpm.input("Nom")
    bpm.number_input("Age")
    bpm.form_submit_button("Envoyer")
