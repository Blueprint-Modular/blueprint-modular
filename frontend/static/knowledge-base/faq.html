FAQ

Questions fréquentes sur Blueprint Modular.

Installation et démarrage

Q : Quelle version de Python ? — Python 3.9 ou supérieur.

Q : Comment lancer mon app ? — bpm run app.py depuis le dossier contenant app.py.

Comportement

Q : Pourquoi l'app se recharge à chaque action ? — BPM, comme Streamlit, réexécute le script à chaque interaction. Utilisez bpm.session_state pour conserver des valeurs.

Q : Différence entre composants natifs et sandbox ? — Les composants natifs sont rendus côté serveur BPM ; certains UIs avancés sont servis via une iframe (sandbox). L'API reste la même pour l'utilisateur.

Déploiement

Q : Peut-on déployer sur un VPS ? — Oui. Voir Deploy — Platforms.

Q : BPM Cloud ? — Non, pas de service cloud dédié pour l'instant.

Voir aussi : Troubleshooting pour le dépannage.
