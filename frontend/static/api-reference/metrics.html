metric

Affiche un indicateur avec un libellé, une valeur et un delta optionnel.

Tester en direct

Modifiez valeur, label et delta en direct sur l'app :

Visuel live et sandbox (modifier valeur, label, delta) sur lapp ?

Signature

bpm.metric(label, value, delta=None, delta_fmt=None, help=None)

Paramètres

ParamètreTypeDéfautDescription
labelstr — Titre de la métrique.
valuenumber | str — Valeur affichée.
deltanumber | NoneNoneVariation (? positif, ? négatif).
delta_fmtstr | NoneNoneFormat d'affichage du delta (ex. devise).
helpstr | NoneNoneInfo-bulle à côté du libellé.

Exemple minimal

import bpm

bpm.metric("Valeur", 142500, delta=3200)
bpm.metric("Taux", "12.5 %", delta=-1.2)

Preview live

Catalogue des composants — Metric, SparklineMetric.

Variantes

Delta inverse : pour un indicateur où une baisse est positive (ex. coût). Devise : passer delta_fmt pour formater le delta.

Variables CSS : --bpm-metric-value-color, --bpm-delta-up, --bpm-delta-down.
