Media

Affichage d'images, audio, vidéo, PDF et logo : bpm.image, bpm.audio, bpm.video, bpm.pdf, bpm.logo.

Mapping Streamlit → BPM

StreamlitBPM
st.image()bpm.image()
st.audio()bpm.audio()
st.video()bpm.video()
st.pdf()bpm.pdf()
st.logo()bpm.logo()

Exemples

import bpm

bpm.image("https://example.com/image.png", caption="Légende")
bpm.audio("sounds/beep.wav")
bpm.video("videos/demo.mp4")
bpm.pdf("doc.pdf")
bpm.logo()  # logo par défaut de l'app
