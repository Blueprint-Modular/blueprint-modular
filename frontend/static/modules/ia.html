Module IA

Interface et intégration IA (Oliver) : chat, complétion, agents. Conçu pour brancher des APIs (Claude, OpenAI, etc.).

Symboles @ et $

Dans le module IA / Oliver :

  • @ — Choix du modèle IA (GPT, Claude, etc.).
  • $ — Choix d'une donnée de l'application : table, chart, metric ou data.

Fonctionnalités prévues

  • Chat avec un modèle (streaming, historique)
  • Complétion / génération de texte (résumés, reformulation)
  • Agents et outils (appels API, recherche)
Note — Ce module est prévu dans la roadmap. Voir Ce qu'il reste à faire.
