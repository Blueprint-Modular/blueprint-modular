Texte

Composants d'affichage de texte. Une page par composant :

  • write — texte brut ou objet Python
  • markdown — contenu Markdown
  • text — texte simple
  • title — titre (niveaux 1 à 4)
  • header — en-tête de section
  • subheader — sous-titre
  • caption — légende
  • code — bloc de code
  • latex — formules LaTeX
  • badge — badge / étiquette
  • html — HTML brut

Voir le catalogue des composants (texte, titre, markdown, code).

Variables CSS : --bpm-font-family, --bpm-color-text, --bpm-color-muted.
