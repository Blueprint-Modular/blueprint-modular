Démarrage en moins d’une minute

Blueprint Modular est un framework d’apps. Installez-le comme n’importe quelle lib Python :

pip install blueprint-modular
bpm run app.py

C’est tout. Ensuite, consultez la documentation et le dépôt GitHub.

Blueprint Modular repose sur trois idées

Écrire du script

Construisez une app en quelques lignes avec une API simple. L’interface se met à jour quand vous sauvegardez le fichier.

Composer l’interface

Ajouter un widget, c’est déclarer une variable. Pas de backend, routes, HTML ou CSS à gérer.

Déployer tout de suite

VPS, Docker, Nginx — la doc décrit les concepts et plateformes pour mettre en production.
