Inputs

Widgets de saisie et d'action : bpm.button, bpm.toggle, bpm.select, bpm.slider, bpm.input, bpm.date_input, etc.

Référence API (sélection)

FonctionBPM
button()bpm.button()
toggle()bpm.toggle()
selectbox()bpm.select()
slider()bpm.slider()
text_input()bpm.input()
number_input()bpm.number_input()
date_input()bpm.date_input()
download_button()bpm.download_button()
file_uploader()bpm.file_uploader()
color_picker()bpm.color_picker()

Exemples

import bpm

bpm.button("Cliquer")
x = bpm.toggle("Activer")
choice = bpm.select("Option", ["A", "B", "C"])
val = bpm.slider("Valeur", 0, 100, 50)
txt = bpm.input("Nom")
n = bpm.number_input("Quantité", 0, 100, 10)
d = bpm.date_input("Date")
bpm.download_button("Télécharger", data=bytes(), file_name="file.csv")
