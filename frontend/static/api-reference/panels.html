Panels

Blocs d'information : bpm.panel avec variantes info, warning, error, success.

Tester en direct

Modifiez variant et titre en direct sur l'app :

Visuel live et sandbox (modifier variant, title) sur l’app →

Signature

bpm.panel(body, variant='info', title=None, icon=True)

Exemples

import bpm

bpm.panel("Message informatif.", variant="info")
bpm.panel("Attention à cette action.", variant="warning")
bpm.panel("Une erreur s'est produite.", variant="error")
bpm.panel("Opération réussie.", variant="success")
bpm.panel("Détails ici.", variant="info", title="À savoir")

Sandbox

Aperçu des variantes bpm.panel :

iInfo
Message informatif.
+Success
Opération réussie.
!Warning

!Error
