Versions

Historique des versions de Blueprint Modular et contenu de chaque version.

v0.1 (actuelle)

  • Package Python publié sur PyPI : pip install blueprint-modular
  • Package installable (pyproject.toml, bpm run / bpm init)
  • Serveur HTTP, premier chargement + interactions boutons (re-run par requête)
  • APIs composants : title, write, markdown, button, metric, table, header, subheader, caption, code, divider, toggle, panel
  • session_state pour persister l'état entre les runs
  • Documentation : Démarrage, Composants, Référence API, Déployer, Base de connaissances, Modules, Versions

À venir

  • v0.2 — WebSocket ou SSE pour réactivité granulaire
  • v0.3 — Multi-langue (site + composant bpm.i18n) ; Carousel et Sidebar dans les composants
  • v0.4+ — Modules pré-construits : Wiki, IA, Veille, IBKR, Analyse de document
