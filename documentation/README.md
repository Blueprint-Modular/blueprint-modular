# Documentation Blueprint Modular

Le site de documentation est un site **statique HTML/CSS**.

Il se trouve à la **racine du projet** :

- **index.html** — Accueil
- **get-started/** — Installation, Fundamentals, First app
- **api-reference/** — Text, Data, Metrics, Charts, Inputs, Layout, Panels, Media, Status, Chat, Config
- **deploy/** — Concepts, Platforms
- **knowledge-base/** — FAQ, Troubleshooting
- **cheat-sheet.html** — Cheat sheet
- **doc.css** — Feuille de style commune (thème BPM, accent #d4af37, dark mode)

Prévisualisation en local : `python -m http.server 8080` à la racine du projet, puis ouvrir http://localhost:8080.

Dépôt : [github.com/remigit55/blueprint-modular](https://github.com/remigit55/blueprint-modular).
